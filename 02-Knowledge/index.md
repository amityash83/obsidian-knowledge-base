# Knowledge Index

## Summary
This note groups durable learning topics by technical area. It is intended to make study, retrieval, and agent-based navigation predictable.

## Concepts
- Knowledge notes should be self-contained and domain-labeled.
- Category indexes improve browsing and retrieval filtering.
- Related-topic links provide context expansion without duplicating content.

## Commands / Code
```bash
# Find all knowledge notes
find /Users/amitmishra/temp/ObsidianVault/02-Knowledge -name "*.md" | sort
```

## Architecture / Flow
1. Open a category section.
2. Jump to a note.
3. Follow related topics for surrounding context.

## Use Cases
- Studying a domain systematically.
- Building a retrieval corpus with stable category anchors.
- Giving MCP agents reliable entry points into the knowledge graph.
