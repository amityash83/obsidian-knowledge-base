## 📌 What are AI Agents?

AI Agents are systems that can:

* Perceive input
* Make decisions
* Take actions

👉 Think:

> LLM + Tools + Memory + Execution = Agent

---

## 🔹 Core Capabilities

* Understand user intent
* Plan steps
* Execute actions
* Learn from feedback

---

## 🔹 Agent Architecture

```text
User Input
   ↓
LLM (Reasoning)
   ↓
Tool Selection
   ↓
Execution
   ↓
Memory Update
```

---

## 🔹 Types of Agents

### 1. Reactive Agents

* No memory
* Single-step actions

---

### 2. Tool-Using Agents

* Call APIs
* Execute scripts

---

### 3. Multi-Step Agents

* Plan + execute workflows

---

### 4. Multi-Agent Systems

* Multiple agents collaborating
