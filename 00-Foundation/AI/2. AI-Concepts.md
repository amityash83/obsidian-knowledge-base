# 🧠 AI Concepts

tags: #AI #Concepts #Learning

---

## 📌 What are AI Concepts?

AI concepts are the **core building blocks** used to design intelligent systems.

They help machines:

* Learn from data
* Make decisions
* Improve over time

---

## 🔹 Machine Learning (ML)

Machine Learning is a subset of AI where systems learn patterns from data.

### Types:

* Supervised Learning
* Unsupervised Learning
* Reinforcement Learning

👉 Related: [[ML vs DL]]

---

## 🔹 Deep Learning (DL)

* Uses neural networks with multiple layers
* Handles complex data like images, audio, text

**Examples:**

* LLMs
* Image recognition
* Speech systems

---

## 🔹 Neural Networks

* Inspired by human brain
* Consists of layers:

  * Input layer
  * Hidden layers
  * Output layer

---

## 🔹 Natural Language Processing (NLP)

* Understanding human language

**Examples:**

* Chatbots
* Translation
* Text summarization

---

## 🔹 Computer Vision

* Understanding images/videos

**Examples:**

* Face detection
* Object detection
* Medical imaging

---

## 🔹 Reinforcement Learning (RL)

* Learning by trial and error
* Uses rewards and penalties

**Examples:**

* Game AI
* Robotics
* Autonomous systems

---

## 🔹 Generative AI

* Creates new content

**Examples:**

* Text (ChatGPT)
* Images (Stable Diffusion)
* Code generation

---

## 🔹 RAG (Retrieval-Augmented Generation)

* Combines:

  * Knowledge retrieval
  * LLM generation

👉 Useful for:

* Chatbots with custom knowledge
* Internal documentation systems

👉 Related: [[RAG Basics]]

---

## 🔹 Embeddings

* Converts text into vectors (numbers)
* Helps AI understand similarity

👉 Used in:

* Search
* RAG systems
* Recommendations

---

## 🔹 Vector Databases

* Store embeddings
* Enable fast similarity search

**Examples:**

* Pinecone
* Weaviate
* FAISS

---

## 🔹 AIOps

* AI for IT Operations

**Use cases:**

* Log analysis
* Anomaly detection
* Incident prediction

👉 Related: [[AIOps Overview]]
