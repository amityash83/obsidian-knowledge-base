# Projects Index

## Summary
This index lists active and reference projects in the vault. It acts as the project navigation hub for human readers, RAG pipelines, and MCP agents.

## Concepts
- Project notes capture objectives, architecture, and execution context.
- Index notes provide stable navigation anchors.
- Consistent linking improves retrieval and reduces orphaned notes.

## Commands / Code

## Architecture / Flow
1. Open a project note.
2. Follow links to knowledge, snippets, and logs.
3. Update the project note as implementation evolves.

## Use Cases
- Starting work from a project-oriented view.
- Mapping project work to reusable knowledge artifacts.
- Providing a curated retrieval entry point.
